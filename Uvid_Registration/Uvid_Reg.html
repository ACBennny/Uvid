Quick Guide


Account Creation

Here's a brief set of instructions to follow when creating your account

1. Username must be minimun of '5' characters but not more than '20' characters and contain only letters, numbers and "one" underscore.

2. Username cannot start/end with an underscore.

3. Username cannot start/ with an number.

4. Password must be minimun of '8' characters but not more than '100' characters.

5. Password must contain at least one of a "letter" , "number" , "special character" in any order.

6. Confirm Password must be the same as the Password

7. Email must be valid as it will be verified before account can be created.

8. Upon processing, a verification code will be sent to the email inputed.
The user will need to input this code before account can be created (serves as email verification).


Account Log In

User is required to input their username and password in order to log into their account.
Upon inability to recall password, it can be reset by correctly inputting the email.

If email cannnot be provided, please Contact Customer Care for further assistance.
